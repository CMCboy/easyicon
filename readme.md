<h3>流程</h3>

检测剪切板变化->检测是否为url->提取ico链接及名字->下载->使用PythonMagick转为ico（可省略）

<h3>使用方法</h3>

运行->复制图片链接即可->python会在本文件夹自动创建ico文件夹，并下载

UI版 双击panda即可关闭程序
